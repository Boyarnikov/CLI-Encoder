# Консольный шифратор

Поддержка 3-x шифров
- Шифр Цезаря
- Шифр Виженера
- Atbash's cipher

methods: easy alphabet inversion

Примеры использования 

```commandline
python main.py encode caesar text.txt
```

```commandline
python main.py encode caesar "my_text"
```

```commandline
python main.py decode caesar text.txt
```
